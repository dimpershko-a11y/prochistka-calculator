# PRO-CHISTKA калькулятор v4.2-text-header

Мини-приложение собрано из рабочего файла `calculator-calculator-1111.html`.

## Главное

- `index.html` — программа калькулятора.
- `assets/app.js` — логика программы. При обновлениях меняется этот файл.
- `assets/style.css` — стили.
- `config.js` — все настраиваемые данные: ставки, коэффициенты, доп. услуги, описания услуг, способы оплаты, бренд, пароль.
- `tilda_iframe_example.html` — пример iframe-кода для Tilda.

## Что вынесено в config.js

- базовые ставки и минималки;
- скорости работ;
- коэффициенты заставленности и загрязнённости;
- список доп. услуг, цены, категории, единицы измерения и время;
- описания “что входит” для каждого вида уборки;
- отдельное описание работ по окнам / остеклению;
- способы оплаты;
- данные компании и PDF-настройки;
- пароль доступа к изменениям.

## Описание окон

Текст `defaults.serviceDescriptions.windows` добавляется в смету и PDF только если в заказе есть выбранные услуги из категории `Окна`.

## Пароль

Пароль теперь не закрывает весь калькулятор. Без пароля можно считать, выбирать услуги, сохранять заказы и формировать смету. Пароль нужен только для изменения тарифов, настроек, доп. услуг, описаний и восстановления резервной копии.

Пароль задаётся в `config.js`:

```js
APP_PASSWORD: '7421'
```

Чтобы отключить пароль, оставьте пустую строку:

```js
APP_PASSWORD: ''
```


## Что изменено в v4.1

- Добавлена кнопка `Сохранить настройки` в настройках и тарифах.
- Пароль теперь открывает только режим изменений, а не весь калькулятор.
- Текст названий выбираемых услуг сделан многострочным, чтобы длинные названия были полностью видны.
- Блок `Описание услуг / что входит в услугу` поднят выше списка доп. услуг в настройках.
- Обновлён cache name в `sw.js`, чтобы браузер подтянул новую версию.

## Обновления

При обновлении программы старайтесь заменять только:

- `index.html`
- `assets/app.js`
- `assets/style.css`
- `manifest.json`
- `sw.js`

Файл `config.js` не трогайте, чтобы не потерять цены, описания и настройки.

## Перенос заказов

В приложении есть кнопки:

- `Экспорт заказов`
- `Импорт заказов`
- `Резервная копия`
- `Восстановить`

Для временной бесплатной версии это замена облачной синхронизации.


## v4.2-text-header

Добавлена текстовая шапка PDF без обязательной картинки-логотипа. В настройках можно менять шрифт, размер названия, жирность, межбуквенное расстояние, цвет и толщину линии. Большой base64-логотип удалён из config.js, чтобы файл был легче.
