# PRO-CHISTKA калькулятор v4.0-config

Мини-приложение собрано из рабочего файла `calculator-calculator-1111.html`.

## Главное

- `index.html` — программа калькулятора.
- `assets/app.js` — логика программы. При обновлениях меняется этот файл.
- `assets/style.css` — стили.
- `config.js` — все настраиваемые данные: ставки, коэффициенты, доп. услуги, описания услуг, способы оплаты, бренд, пароль.
- `tilda_iframe_example.html` — пример iframe-кода для Tilda.

## Что вынесено в config.js

- базовые ставки и минималки;
- скорости работ;
- коэффициенты заставленности и загрязнённости;
- список доп. услуг, цены, категории, единицы измерения и время;
- описания “что входит” для каждого вида уборки;
- отдельное описание работ по окнам / остеклению;
- способы оплаты;
- данные компании и PDF-настройки;
- пароль входа.

## Описание окон

Текст `defaults.serviceDescriptions.windows` добавляется в смету и PDF только если в заказе есть выбранные услуги из категории `Окна`.

## Пароль

Пароль задаётся в `config.js`:

```js
APP_PASSWORD: '7421'
```

Чтобы отключить пароль, оставьте пустую строку:

```js
APP_PASSWORD: ''
```

## Обновления

При обновлении программы старайтесь заменять только:

- `index.html`
- `assets/app.js`
- `assets/style.css`
- `manifest.json`
- `sw.js`

Файл `config.js` не трогайте, чтобы не потерять цены, описания и настройки.

## Перенос заказов

В приложении есть кнопки:

- `Экспорт заказов`
- `Импорт заказов`
- `Резервная копия`
- `Восстановить`

Для временной бесплатной версии это замена облачной синхронизации.
