# PRO-CHISTKA Calculator Mini-App v3.0-free

Бесплатная временная версия калькулятора для публикации через GitHub Pages.

## Что внутри

- `index.html` — основная страница калькулятора.
- `assets/style.css` — стили.
- `assets/app.js` — логика калькулятора.
- `config.js` — пароль входа.
- `manifest.json` и `sw.js` — PWA-заготовка для установки на телефон.
- `tilda_iframe_example.html` — пример iframe-кода для Tilda.

## Как работает хранение

В этой версии нет облачной базы. Данные сохраняются в браузере через `localStorage`.

Для переноса между телефоном и ПК используйте:

- `Экспорт заказов` — скачивает JSON только с заказами.
- `Резервная копия` — скачивает JSON со всеми настройками, тарифами, услугами и заказами.
- `Импорт` — загружает ранее скачанный JSON.

## Пароль

Пароль задаётся в `config.js`:

```js
window.PROCHISTKA_CONFIG = {
  APP_PASSWORD: '7421',
  SHOW_AUTH_CARD_AFTER_LOGIN: false
};
```

Поменяйте `7421` на свой пароль перед публикацией.

Важно: это клиентский пароль для временной защиты от случайного доступа. Настоящую защиту логин/пароль лучше делать на сервере: VPS + PocketBase.

## Публикация через GitHub Pages

1. Создайте репозиторий на GitHub, например `prochistka-calculator`.
2. Загрузите все файлы из этой папки в корень репозитория.
3. Откройте `Settings → Pages`.
4. Включите публикацию из ветки `main`, папка `/root`.
5. После публикации GitHub даст ссылку вида:

```text
https://USERNAME.github.io/prochistka-calculator/
```

## Вставка в Tilda

Создайте скрытую страницу на Tilda, поставьте пароль страницы и вставьте блок T123 с iframe-кодом из `tilda_iframe_example.html`.

Замените `PASTE_GITHUB_PAGES_URL_HERE` на ссылку GitHub Pages.

## Обновление версии

После изменения файлов в GitHub обновите страницу калькулятора. Если браузер показывает старую версию, очистите кэш сайта или откройте в режиме инкогнито.
